RSI 相对强弱指标 Relative Strength Index

作用：通过一段时间内上涨幅度与下跌幅度的比值来衡量价格的相对强弱，常用于判断超买超卖。RSI > 80 视为超买区，RSI < 20 视为超卖区。

计算公式（Wilder 平滑）
Δ = close[i] - close[i-1] gain = max(0, Δ) loss = max(0, -Δ) avg_gain[i] = (avg_gain[i-1] * (n-1) + gain) / n avg_loss[i] = (avg_loss[i-1] * (n-1) + loss) / n RS = avg_gain / avg_loss RSI = 100 - 100 / (1 + RS)

MACD 指数平滑异同移动平均线 Moving Average Convergence Divergence

作用：通过快慢 EMA 的差值（DIF）衡量趋势动能，再对 DIF 做 EMA 平滑得到信号线（DEA）。DIF 上穿 DEA 为金叉买入信号，下穿为死叉卖出信号；柱状图反映两者差距。

计算公式
EMA_fast = EMA(close, fast_period) EMA_slow = EMA(close, slow_period) DIF = EMA_fast - EMA_slow DEA = EMA(DIF, signal_period) MACD 柱 = (DIF - DEA) × 2

布林带 Bollinger Bands BOLL

作用：以 N 日均线为中轨，上下各加减 K 倍标准差形成通道。价格突破上轨可能见顶、跌破下轨可能见底；带宽收窄（< 5%）预示即将变盘。

计算公式
中轨 MID = MA(close, N) 标准差 σ = Std(close, N) 上轨 UPPER = MID + K × σ 下轨 LOWER = MID - K × σ 带宽 = (UPPER - LOWER) / MID × 100%

KDJ 随机指标 扩展指标

作用：综合了动量、强弱和移动平均线的优点，常用于短线买卖点判断。K、D 值在 20 以下金叉视为超卖反弹机会，80 以上死叉视为超买回调风险；J 值 > 100 或 < 0 提示短期极端状态。

计算公式
RSV = (close - lowest_low_N) / (highest_high_N - lowest_low_N) × 100 K = (M1-1)/M1 × K_prev + 1/M1 × RSV D = (M2-1)/M2 × D_prev + 1/M2 × K J = 3K - 2D
